"Understanding Inequities in participation in Energy Efficiency Programs for the Public Sector"
This paper, through the Clean Energy Transformation Lab at UCSB, analyzes the disparities in public sector energy efficiency program participation throughout California. My focus was on city level data, using multivariate polynomial regression on demographic variables to understand determining factors in clean energy program participation.

The manuscript has been finalized, but this paper has not been published yet.

Live Manufacturing and Safety Dashboards
I designed and built a multi-page web hosted dashboard app for live manufacturing status, progress, efficiency, and safety data. The app was written in Python, with HTML and CSS for UI, and it contained 4 different dashboards. SQL connection libraries were used to query local DB's, and external data was accessed through REST API's. I later migrated the project to Power BI as the company transitioned to the MS Suite. This project was during an internship at Atomica in Santa Barbara, CA.
Desktop Application for Audio Sample Management
For my own personal use, I designed and wrote an Electron app to congregate audio samples in multiple formats and apply automatic organization through custom category tags. Designed to be used simultaneously with any audio software, this application allows for searching by name or tag/category and drag-and-drop functionality of any indexed sample to any application. The app was written in Node.JS with HTML and CSS for UI.

This was a personal project to build JavaScript, HTML, and CSS skills.
